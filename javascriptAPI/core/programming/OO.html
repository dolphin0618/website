描述

new只有两个作用 1.替你创建Object，给this 2.替你返回this

描述

判断是何对象(比较构造函数)：
var a=new Array();
alert(a.constructor==Array);// true

var a=new Array();
alert(a.constructor==Object);//false
利用apply实现继承：
function Person(name, age)
{
	this.name=name;
	this.age=age;
}
function Worker(name, age, job)
{
	Person.apply(this, arguments);//改变this
	this.job=job;
}

Worker.prototype=new Person();//让worker的原型指向新实例的父类
Worker.prototype.constructor=Worker;//并改变其构造器指向

Worker.prototype.showJob=function ()
{
	alert(this.job);
};

var w=new Worker('blue', 25, '打杂的');
