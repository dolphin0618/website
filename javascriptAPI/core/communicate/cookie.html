Cookie

描述

临时存东西:

1. cookie 必须放到服务器环境，以域为单位

2. 容量 (4K)

3. 不安全

4. 取出来都是字符串

5. 不能跨浏览器

列表

状态 返回值说明
document.cookie 获取所有cookie

存cookie: document.cookie='名字=值'
expires 到期时间，默认session(会话)，关了浏览器自动删除 document.cookie='name=value;expires='+oDate;
path

路径 默认当前文件夹

path=/ 根

document.cookie='name=value;path=/;expires='+oDate;

cookie封装

cookie添加js代码：
function addCookie(name,value,iDay){
	if(iDay){
		var oDate=new Date();
		oDate.setDate(oDate.getDate()+iDay);
		
		document.cookie=name+'='+value+';path=/;expires='+oDate;
	}else{
		document.cookie=name+'='+value+';path=/';
	}
}
cookie获取js代码：
function getCookie(name){
	var arr=document.cookie.split('; ');
	
	for(var i=0; i<arr.length; i++){
		var arr2=arr[i].split('=');
		//arr2[0]  name
		//arr2[1]  value
		if(name==arr2[0]){
			return arr2[1];	
		}
	}
	return '';
}
cookie删除js代码：
function removeCookie(name){
	//alert('首先先f12->然后resource->cookie->');
	addCookie(name,'1',-10);
}
