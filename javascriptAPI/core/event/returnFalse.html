默认事件

描述

事件默认行为：当一个事件发生的时候浏览器自己会默认做的事情

事件传播和默认事件是二个不同的机制；一个是节点之间传递事件 一个是执行默认操作

阻止默认事件行为方法

方法 说明
return false 阻止默认事件，对于用addEventListener绑定事件的默认事件无法阻止；在IE678中不能阻止选中文字的默认事件，可以用鼠标捕获解决
event.preventDefault() 阻止默认事件，对于用addEventListener绑定事件的默认事件可以阻止；不兼容IE678

示例

描述

js代码：
document.addEventListener('contextmenu',function(){
	return false;//阻止默认事件无效
	event.preventDefault();//有效果
},false);
