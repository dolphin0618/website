事件委托

描述

子元素的事件委托给父级，并可以通过事件源找到具体子元素

列表

事件源获取方法 支持情况
oEvent.srcElement 支持IE Chrome
oEvent.target 支持FF

示例

描述

事件委托的用途举例

html代码：
<ul id="ul1">
	<li></li>
	<li></li>
	<li></li>
</ul>
js代码：
var oUl=document.getElementById('ul1');
oUl.onclick=function(ev){
	var oEvent=ev || event;
	//兼容处理
	var target=oEvent.srcElement || oEvent.target;
	//target即是ul中被点击的子元素
	if(target.tagName=='LI'){
		target.style.background='green';
	}
};
//可以利用此机制给新创建的元素加事件，如jquery中的live方法
