鼠标捕获

描述

鼠标捕获：只在IE里面存在；常用来处理IE6-8选中文字问题

当一个object的被 setCapture 后，他的方法将会被继承到整个文档进行捕获并不向子元素传递，即捕获后其它所有事件会失效。当不需要把方法继承到整个文档捕获时，要用 object.releaseCapture 来释放.

参数

方法 说明
setCapture() 设置捕获
relaseCapture() 释放捕获

示例

描述

当前元素设置捕获后，其它所有元素事件失效；当前元素释放捕获后，其它所有元素事件恢复；

js代码：
oDiv.setCapture && oDiv.setCapture();//给oDiv设置捕获
oDiv.releaseCapture && oDiv.releaseCapture();//让oDiv释放捕获
