事件绑定

描述

可以对同一事件绑定多次，避免冲突

DOM开头的事件只能用绑定来加，如DOMContentLoaded，DOMMouseScroll等

参数

方法 说明
addEventListener 事件绑定：obj.addEventListener(事件名，函数名,boolean); 事件名不带on；第3个参数设置事件是在事件捕获时执行，还是事件冒泡时执行true=捕获|false=冒泡（支持IE9+、chrome、FF）
attachEvent 事件绑定：obj.attachEvent(事件名,函数名)；（支持IE）
removeEventListener 解除绑定：obj.removeEventListener(事件名,函数名,false);没法解绑定匿名函数（支持IE9+、chrome、FF）
detachEvent obj.detachEvent(事件名，函数名);没法解绑定匿名函数（支持IE）

示例

描述

绑定事件js兼容代码：
function fn(ev){};//不管哪种方法，ev都是当前事件对象
if (window.addEventListener) { 
	window.addEventListener('load',fn,false); 
} else { 
	window.attachEvent('onload',fn); 
}
解除绑定js兼容代码：
if (window.removeEventListener) { 
	window.removeEventListener('load',fn,false); 
} else { 
	window.detachEvent('onload',fn); 
}
