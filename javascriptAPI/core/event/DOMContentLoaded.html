DOMContentLoaded

描述

1.监听页面加载事件（不兼容IE678）

2.加载页面只要dom加载完毕就执行，要比onload快（onload在页面所有dom和图片等资源加载完才执行）

参数

示例

描述

页面加载，比onload要快。

js兼容代码：
if(document.addEventListener){
	document.addEventListener('DOMContentLoaded',function(){ //IE9+ chrome FF
		//code...
	},false);
}else{  //IE 6 7 8
	document.attachEvent('onreadystatechange',function(){
		if(document.readyState=='complete'){
			//code...
		}	
	});
}
