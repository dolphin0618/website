字数统计

描述

监听文本输入框内容变化，常用来做字数统计

示例

描述

兼容写法。

js代码：
var oT=document.getElementById('input');
//oninput:兼容IE10 chrome FF; 
//onpropertychange:兼容IE系;
oT.oninput=oT.onpropertychange=function(){
	document.title=oT.value.length;
};
