currentStyle 返回值: <String>

描述

Element.style属性可以获得html标签的样式，但是不能获取非行间样式.而currentStyle和getComputedStyle可以

获取元素的最终样式，只适用于IE系列。不适用获取复合样式，如background

示例

描述

获取name样式的值。

js代码：
Element.currentStyle.name
Element.currentStyle[name]
处理兼容js代码：

if(Element.currentStyle){
	//currentStyle
}else{
	//getComputedStyle
}
