offset 返回值: <String>

描述

获取最终显示的样式，offsetTop、offsetLeft、offsetParent与css定位有关

参数

属性 说明
offsetWidth 实际宽度 + padding + border
offsetHeight 实际高度 + padding + border
offsetTop 与最近的并且有定位的父级的上边距
offsetLeft 与最近的并且有定位的父级的左边距
offsetParent 获取到最近的并且有定位的父级
