client 返回值: <String>

描述

获取可视区的样式，clientLeft,clientTop这两个返回的是元素周围边框的厚度,如果不指定一个边框或者不定位改元素,他的值就是0.

属性

属性 说明
clientWidth 实际宽度 + padding
clientHeight 实际高度 + padding
clientTop 返回对象的offsetTop属性值和到当前窗口上边的真实值之间的距离（不常用）
clientLeft 返回对象的offsetLeft属性值和到当前窗口左边的真实值之间的距离（不常用）

示例

描述

浏览器宽度js代码：
document.documentElement.clientWidth || document.body.clientWidth;
