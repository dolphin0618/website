scrollTop 返回值: <String>

描述

元素中内容 的高度会超过 元素本身 的高度, scrollTop指的是 元素中的内容 超出 元素上边界 的那部分的高度。也就是滚动过的距离。scrollTop是可读写属性。

兼容性

获取整个页面最外层滚动条的scrollTop 兼容情况
document.body.scrollTop 兼容 chrome
document.documentElement.scrollTop 兼容 IE系 FF

示例

描述

js代码
//获取
Element.scrollTop;
//设置
Element.scrollTop = 0;

获取最外层滚动条的scrollTop兼容处理

js代码
var scrollT=document.documentElement.scrollTop || document.body.scrollTop;
