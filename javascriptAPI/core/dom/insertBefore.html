insertBefore

描述

在已有的子节点前插入一个新的子节点。并返回新的子节点。

示例

描述

html代码：
<ul>
  <li>111</li>
  <li>222</li>
</ul>
js代码：
var obj1 = oUl.children[0];
var obj2 = oUl.children[1];
oUl.insertBefore(obj2, obj1);//把obj2插入到obj1的前面
结果：
<ul>
  <li>222</li>
  <li>111</li>
</ul>
