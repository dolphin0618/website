cloneNode

描述

此方法用来克隆元素

方法参数为true，则克隆oldElement 这个及其子节点，否则只克隆这个节点本身。

示例

描述

html代码：
<ul class="abc">
  <li>111</li>
</ul>
js代码：
var obj = oUl.cloneNode(true);
document.appendChild(obj)
结果：
<ul class="abc">
  <li>111</li>
</ul>
<ul class="abc">
  <li>111</li>
</ul>
js代码：
var obj = oUl.cloneNode(false);
document.appendChild(obj)
结果：
<ul class="abc">
  <li>111</li>
</ul>
<ul class="abc"></ul>
