基本选择器 <dom>

描述

原生js中的基本选择器

列表

方法 简介
getElementById([ID]) 通过ID来选择具体某一个元素
getElementsByName([name]) 通过name来选择一组元素
getElementsByTagName([tagName]) 通过元素名来选择一组元素

示例

描述

用法：

html代码：
<ul>
	<li id="li1" name=""></li>
	<li id="li2" name="lis"></li>
	<li id="li3" name="lis"></li>
</ul>
js代码：
var oLi = document.getElementById('li1');
//oLi是获取文档中id为li1的元素对象，即第一个li

var oLi1 = document.getElementsByName('lis');
//oLi1是获取文档中name为lis的一组元素对象，即第二和第三个li

var oLi2 = document.getElementsByTagName('li');
//oLi2是获取文档中标签名为li的一组元素对象，即所有li
