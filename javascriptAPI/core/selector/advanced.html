高级选择器 <dom>

描述

原生js中的高级选择器，适用于支持HTML5的高级浏览器

列表

方法 简介
getElementsByClassName([class]) 通过class来选择一组元素
querySelectorAll([css选择器规则]) 通过css选择器规则来选择一组元素
querySelector([css选择器规则]) 通过css选择器规则来选择一个元素

示例

描述

用法：

html代码：
<ul class="uls">
	<li class=""></li>
	<li class="lis"></li>
	<li class="lis"></li>
</ul>
js代码：
var oLi = document.getElementsByClassName('lis');
//oLi是获取文档中class为lis的一组元素对象，即第二和第三个li

var oLi1 = document.querySelectorAll('.uls li');
//oLi1是获取文档中ul下所有li

var oLi2 = document.querySelector('.uls li');
//oLi2是获取文档中ul下的第一个li
