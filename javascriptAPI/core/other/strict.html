严格模式

描述

'use strict' 有作用范围，而且在不同的js文件范围属于当前文件中

建议加上严格模式，每个JS文件都加

在严格模式下会出错的地方
没有var时会报错
with无法使用
function不能在if、for里面定义
this 严格模式的范围： 1.函数 2.script标签

示例

描述

严格模式作用域：
<script>
　　"use strict";
　　console.log("这是严格模式。");
</script>
<script>
　　console.log("这是正常模式。");
</script>
function strict(){
　　"use strict";
　　return "这是严格模式。";
}
function notStrict() {
　　return "这是正常模式。";
}
在eval作用域：
"use strict";
var x = 2;
console.info(eval("var x = 5; x")); // 5
console.info(x); // 2
