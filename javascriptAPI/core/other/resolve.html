预解析

描述

把变量(函数定义)的声明提前，赋值还在原地

预解析作用范围：1. 冲不出script 2. 函数内部

示例

描述

样例。

函数和变量的预解析：
//js是在调用函数之前就将函数放入内存中
show();  //hellow
function show(){
  alert("hellow");
}

//变量的声明也在一开始就被放入内存中，但是并没有赋值
alert(a);//undefined
var a="hellow";
变量和函数重名：
//重名时，函数名优先级高于变量名
alert(a); //function(){ return  "function"}  
function a(){ return "function" }
var a="var";
预解析不会跨script去解析：
<script type="text/javascript">
alert(a);//报错
</script>

<script type="text/javascript">
var a=123;
</script>
