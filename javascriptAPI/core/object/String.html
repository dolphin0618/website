String

描述

数组对象：var str=new String([val]);

列表

方法 使用说明
length 长度
charAt(index) 取第index位的字符,index超出或者等于length时 返回''
substring substring(开始位置，结束位置)。如开始位置小于结束位置会自动调换
split() 切字符串 字符串->数组
indexOf() 查找子字符串
lastIndexOf() 从后向前搜索字符串
replace() 替换
toUpperCase() 大写
toLowerCase(); 小写
slice(); 提取字符串的片断，并在新的字符串中返回被提取的部分。开始位置必须小于结束位置
trim() 去除首位空格
charCodeAt([index]) 字母编码
String.fromCharCode([code]) 打印出所有汉字：for(var i=0x4e00; i<0x9fa5; i++){ document.write(String.fromCharCode(i)); }
可以使用正则的方法↓↓
match() 找到一个或多个正则表达式的匹配。
replace() 替换与正则表达式匹配的子串。
search() 检索与正则表达式相匹配的值
split() 正则表达式来切分。
