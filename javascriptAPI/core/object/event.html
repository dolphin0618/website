event

描述

event 兼容IE系，chrome

参数 ev 兼容FF、chrome、IE9+

处理兼容：var oEvent=ev || event;

参数

属性 说明
clientX oEvent.clientX：获取当前事件鼠标位置的横向坐标
clientY oEvent.clientY：获取当前事件鼠标位置的纵向坐标
keyCode oEvent.keyCode：监听键盘事件按下的键码
ctrlKey oEvent.ctrlKey：监听ctrl键
shiftKey oEvent.shiftKey：监听shift键
altKey oEvent.altKey:监听alt键
wheelDelta oEvent.wheelDelta：滚动方向IE Chrome -120 下 详见滚轮事件
detail oEvent.detail:滚动方向 FF 3 下 详见滚轮事件
srcElement 事件源 兼容IE系 chrome
target 事件源 兼容FF
toElement 去哪了 兼容IE系，chrome
fromElement 从哪里来？ 兼容IE系，chrome
relatedTarget 对于 mouseover 事件来说，该属性是鼠标指针移到目标节点上时所离开的那个节点。 对于 mouseout 事件来说，该属性是离开目标时，鼠标指针进入的节点。（不兼容IE）
preventDefault oEvent.preventDefault()阻止默认事件，绑定的事件return false无效 兼容IE9+ chrome FF
stopPropagation oEvent.stopPropagation()阻止冒泡， 兼容IE9+ chrome FF

示例

描述

onmouseover的BUG问题。

js代码：
oDiv.onmouseover=function(ev){
	var oEvent=ev || event;
	var from=oEvent.fromElement || oEvent.relatedTarget;
	if(oDiv.contains(from))return;
	alert('移入了');
};
