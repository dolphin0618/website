window对象

列表

方法 使用说明
window.navigator.userAgent 浏览器的信息
window.open window.open(地址，打开方式);window.open(地址); 新窗口打开;返回值是新窗口window对象; chrome 默认屏蔽 FF 阻止了 IE 默认出来了
window.close() chrome: 直接关闭 FF: 没反应 IE： 友好提示*用户能关闭自己open出来窗口
window.location 获取当前地址(url);返回object;
window.location.href 设置或返回当前地址;window.location.href='http://www.baidu.com'; 直接跳转到百度，覆盖自己页面
window.location.search 设置或获取问号后面数据
window.location.hash 设置或获取#号后面数据;赋值时不会刷新页面;锚点
window.location.host 设置或返回主机名和当前 URL 的端口号
window.location.hostname 设置或返回当前 URL 的主机名。
window.location.reload() 重新加载当前文档
