Image

描述

Image对象;var oImg=new Image();通常用来做图片预加载;

缓存：浏览器对同一个图片下载地址(url),只会读一次

列表

方法 使用说明
onload oImg.onload 图片加载成功会调用此方法
onerror oImg.onerror 图片加载失败会调用此方法

示例

描述

图片预加载

js代码：
var oImg=new Image();
oImg.onload = function(){
	//code...
}
oImg.onerror = function(){
	//code...
}
oImg.src = http://...jpg;

* 建议oImg.src写在onload和onerror之后。否则IE8下方法无效；不考虑IE8-可忽略此建议
