正则

描述

一套筛选字符串的规则;

var reg=new RegExp(...,选项);

var reg=/.../选项;

参数

支持正则的方法 说明
reg.test(str) 如果校验成功，返回的true，否则返回false，只要有一部分复合要求，就返回true
str.replace(正则，谁); 替换
str.replace(正则,function(){}) 替换 见下例
str.match()
str.search(正则)

列表

修饰 说明
^ 代表是整个字符串开头 跟m配合，认识字符串中换行
$ 结尾
表达式 说明
[abc] 任意一个 a|b|c
[a-z] 区间
[^abc] 排除
. 任意东西
选项 说明
i ignore 忽略大小写
g global 全局匹配
m multi-line 多行
量词 说明(量词只管前面一个)
+ 最少1个，最多随便
* 可有可无，有了也不限 {0,}
? 可有可无 {0,1}

{n} 代表正好n个 /\d{2}/ /\d\d/

{n,m} 最少n个，最多m个 /\d{2,4}/ /\d{1,1000}/

{n,} 最少n个，最多不限 /\d{2,}/ /\d{1,}/ /\d{0,}/

{,m} 最少不限，最多m个 错的 ×

{0,m}

{0,1}
转义 说明
\d digital 数字 [0-9]
\w word 单词 [a-z0-9_]
\s sapce 空白
\D 非数字 [^0-9]
\W 非单词 [^a-z0-9_]
\S 非空白
\b boundary 单词边界

示例

描述

样例。

座机号码：
/(0[1-9]\d{1,2}-)?[1-9]\d{6,7}/
邮箱：
/\w+@[a-z0-9\-]+(\.[a-z]{2,6}){1,2}/

方法。

捡数字：
var str='abc12 45 678 90 a23 pop90 19 c34';
console.log(str.match(/\d+/g));
结果：
12 45 678 90 23 90 19 34
第一个数字出现的位置：
var str='3W2heel3Delta';
var reg=/3/;

console.log(str.search(reg));
结果：
0
过滤：

var str='aabbccddeeff';
var reg=/bb|ee/g;
document.write(str.replace(reg,function(s){
	var tmp='';
	for(var i=0; i<s.length; i++){
		tmp+='*';
	}
	return '<span style="color:red">'+tmp+'</span>';
}));
结果：
<span style="color:red">aa**ccdd**ff</span>
