Array

描述

数组对象：var arr=new Array([num]);

列表

方法 使用说明
push() 将新元素添加到一个数组中，并返回数组的新长度值
pop() 移除数组中的最后一个元素并返回该元素
shift() 移除数组中的第一个元素并返回该元素
unshift() 将指定的元素插入数组开始位置并返回该数组
splice() 从一个数组中移除一个或多个元素，如果必要，在所移除元素的位置上插入新元素，返回所移除的元素
split([char]) 把字符串用字符(char)切成数组
arr.join([char]) 把数组用字符(char)连接成字符串
arr.concat(arr2,arr3,arr4....) 数组的连接
arr.reverse() 返回一个元素顺序被反转的Array对象；注意这里是返回一个原来的数组而不是创建了一个新的数组
arr.sort(); arr.sort(function(n1,n2){ //比较函数 return n1-n2; }); //从小到大
arr.length 获取数组的长度;
如何快速清空数组？ arr.length=0; arr.splice(0,arr.length); while(arr.length){ arr.pop(); } arr=[];
toString(); 把数组转换为字符串，并返回结果。
万能操作： arr.splice(开始位置,删除长度,元素1，元素2.....);
删除 arr.splice(开始位置,删除长度);
添加 arr.splice(开始位置,0,'a','b');
替换 arr.splice(开始位置,2,'a','b');
