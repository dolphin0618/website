instanceof 返回值: <Boolean>

描述

可以用 instanceof 来判断一个对象是不是某个类或其子类的实例(利用继承关系来判断，跟原型有关)

示例

描述

js代码：
function Foo(){} 
Foo.prototype = new Aoo();
var foo = new Foo(); 
console.log(foo instanceof Foo)//true

null是object类型，但不是object的实例

js代码：
console.log(typeof null);//object
console.log(null instanceof Object);//false

instanceof跟原型的关系

js代码：
function Big(){
	this.name = '猪';
}
var big = new Big();
Big.prototype = {};//在创建一个big之后，改变Big原型
alert(big instanceof Big);//false 这里big不再是Big的实例
