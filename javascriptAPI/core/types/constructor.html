constructor 返回值: <Function>

描述

设置或获取创建对象的构造方法。

根据比较也可以用来判断对象类型。

示例

描述

js代码：
function Big(){
	this.name = '猪';
}
var big = new Big();
//Big.prototype = {};
alert(big.constructor);//Big 这里自己没有会去原型上去找
alert(Big.prototype.constructor);//Big

//在prototype中有二个属性 一个是constructor ，另外一个是__proto__

判断对象类型

js代码：
var arr = [];
console.log(arr.constructor == Array);//true
